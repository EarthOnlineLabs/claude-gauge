What it can and can't see它能看到什么，看不到什么

Built to be auditable天生可审计

It's all readable bash and Python — nothing obfuscated, nothing hidden.全是能读懂的 bash 和 Python——没有混淆，没有藏私。

Token-only access只取 token

Reads one OAuth token from the keychain entry Claude Code-credentials and sends it only to Anthropic, only to fetch usage.只从钥匙串条目 Claude Code-credentials 读取一个 OAuth token，且只发往 Anthropic，只用来拉取用量。

Never reads your chats绝不读你的对话

It never opens ~/.claude/projects, your conversation logs, or any code file. Usage numbers only.绝不打开 ~/.claude/projects、你的对话记录或任何代码文件。只看用量数字。

Open & auditable开源、能审

Three short scripts in plain bash and Python — no minification, no binaries, no telemetry. Read every line before you run it.三个简短的脚本，纯 bash 和 Python——不压缩、无二进制、零埋点。跑之前每一行你都能读。

MIT licensedMIT 许可

Free to use, fork, and modify. Uninstall leaves your Claude Code credentials and data completely untouched.可自由使用、fork 和修改。卸载时绝不动你的 Claude Code 凭证和数据。

Get it running让它跑起来

Two commands and you're done两条命令就搞定

Clone the repo and run the installer. It handles SwiftBar, the background refresher, and the first data pull.克隆仓库，运行安装脚本。SwiftBar、后台刷新器和首次拉取数据，它全包了。

terminal — zsh 
# clone and install — macOS only克隆并安装 —— 仅限 macOS
git clone https://github.com/EarthOnlineDev/claude-gauge.git
cd claude-gauge && ./install.sh

Needs macOS, a signed-in Claude Code with a Pro/Max subscription, and system python3. Optional: point Claude Code's statusLine at the bridge for zero-cost instant updates while you work. Uninstall anytime with ./uninstall.sh — it never touches your Claude Code credentials or data.需要 macOS、已登录且拥有 Pro/Max 订阅的 Claude Code，以及系统自带的 python3。可选：把 Claude Code 的 statusLine 指向桥接脚本，干活时零成本即时刷新。随时用 ./uninstall.sh 卸载——绝不动你的 Claude Code 凭证和数据。

How it works工作原理

Three small layers三个小分层

Each does one job. The render layer never touches the network when the background layer is healthy.每层只干一件事。后台层健康时，渲染层从不碰网络。

01 RENDER渲染

The gauge表盘

plugin/claude-gauge.15s.sh

A SwiftBar plugin that wakes every 15 seconds, reads whichever of live.json or cache.json is newer, and draws the status-aware title and dropdown. Pure rendering — no secrets, no API calls in the happy path.一个 SwiftBar 插件，每 15 秒醒来一次，读取 live.json 和 cache.json 中较新的那个，绘制随状态变色的标题和下拉。纯渲染——正常路径下不碰任何密钥，不发任何 API 请求。

02 DATA数据

The refresher刷新器

refresher/claude-gauge-refresh.sh

A LaunchAgent (dev.earthonline.claude-gauge) fires every 30s and adaptively throttles — 45s when critical, 240s when idle to dodge 429s. It polls the usage API, atomically writes the cache, self-heals the token, and sends threshold notifications.一个 LaunchAgent（dev.earthonline.claude-gauge）每 30 秒触发一次，并自适应节流——告急时 45 秒，闲置时 240 秒，以躲开 429 限流。它轮询用量 API，原子地写入缓存，自动给 token 续命，并在跨过阈值时发送通知。

03 BRIDGE桥接 optional可选

The statusLinestatusLine 桥接

bridge/claude-gauge-statusline.py

Wired as Claude Code's statusLine command, it reads the rate limits CC already hands its status bar and writes live.json — so the menu bar updates instantly while you work, with no API call and no token needed. Applies to new sessions started after you configure it.作为 Claude Code 的 statusLine 命令接入，它读取 CC 已经喂给自己状态栏的限额，写入 live.json——这样你干活时菜单栏即时更新，无需任何 API 请求，也不用 token。只对配置之后新开的会话生效。

Why it's different它有何不同

Small, honest, and out of your way小巧、诚实、不挡道

It only speaks up when it matters — and when it does, it tells the truth.只在要紧时出声——出声时，说的都是实话。

Status-aware minimalism随状态变色的极简主义

When you have headroom it's silent — just the current 5-hour percentage, auto black/white to match your menu bar. Cross a threshold and it surfaces the window that's actually biting, with a color and a reset countdown.宽裕时它安静无声——只显示当前 5 小时窗口的百分比，自动黑白配色匹配你的菜单栏。跨过阈值，它就把真正告急的那个窗口冒出来，带上颜色和重置倒计时。

Self-healing tokenstoken 自动续命

Claude Code doesn't refresh its keychain token while idle, so the usage API starts returning 401. ClaudeGauge spots a near-expired token and runs one tiny headless claude -p ok from /tmp to renew it — so the gauge keeps working even when you've stepped away.Claude Code 闲置时不会刷新钥匙串里的 token，于是用量 API 开始返回 401。ClaudeGauge 发现 token 快过期，就在 /tmp 里跑一条极小的无头命令 claude -p ok 把它续上——这样你离开座位时，表盘照样工作。

Honestly stale过期就老实说

If data is more than 15 minutes old, the title goes gray and grows a ~, and the dropdown warns you. No tool lies to you with old numbers dressed up as live — ever.数据超过 15 分钟没刷新，标题就变灰、长出一个 ~，下拉里也会提醒你。绝不拿过期数字伪装成实时来糊弄你——永远不会。

Notch-friendly刘海友好

On notched Macs the menu bar title stays under ~90px (about 11 characters). Go wider and the notch swallows your icon whole — so the display is engineered to stay short no matter the state.带刘海的 Mac 上，菜单栏标题始终压在约 90px（约 11 个字符）以内。再宽一点，刘海就会把你的图标整个吞掉——所以无论什么状态，显示都被刻意控制得够短。

Knows before you do比你先知道

A background agent polls on an adaptive schedule and fires a native macOS notification the moment you cross 75% or 90% — once per window, per cycle, so it nudges without nagging.一个后台任务按自适应节奏轮询，在你刚跨过 75% 或 90% 的那一刻，弹出原生 macOS 通知——每个窗口、每个周期只提醒一次，点到为止，绝不唠叨。
