What it can and can't see它能看到什么，看不到什么

Private by design绝对隐私安全

Most usage trackers tally your usage by reading your conversation logs — this one never reads a single word.多数用量工具靠翻你的对话记录算用量，它一个字都不碰

Token-only access只取 token

Reads one OAuth token from the keychain entry Claude Code-credentials and sends it only to Anthropic, only to fetch usage.只从钥匙串条目 Claude Code-credentials 读取一个 OAuth token，且只发往 Anthropic，只用来拉取用量。

Never reads your chats绝不读你的对话

The popular usage trackers read ~/.claude/projects — your conversation logs — to add up usage. ClaudeGauge never opens them, or any code file. It asks Anthropic for your usage numbers and nothing else.那些流行的用量工具会去读 ~/.claude/projects——你的对话记录——来累加用量。ClaudeGauge 绝不打开它们，也不碰任何代码文件。它只向 Anthropic 要你的用量数字，别的一概不要。

Open & auditable开源、能审

Three short scripts in plain bash and Python — no minification, no binaries, no telemetry. Read every line before you run it.三个简短的脚本，纯 bash 和 Python——不压缩、无二进制、零埋点。跑之前每一行你都能读。

MIT licensedMIT 许可

Free to use, fork, and modify. Uninstall leaves your Claude Code credentials and data completely untouched.可自由使用、fork 和修改。卸载时绝不动你的 Claude Code 凭证和数据。

Get it running让它跑起来

Two commands and you're done两条命令就搞定

Clone the repo and run the installer. It handles SwiftBar, the background refresher, and the first data pull.克隆仓库，运行安装脚本。SwiftBar、后台刷新器和首次拉取数据，它全包了。

terminal — zsh 
# clone and install — macOS only克隆并安装 —— 仅限 macOS
git clone https://github.com/EarthOnlineDev/claude-gauge.git
cd claude-gauge && ./install.sh
1. Installs SwiftBar via Homebrew用 Homebrew 装好 SwiftBar 2. Loads a 30s background refresher加载一个 30 秒后台刷新器 3. Pulls your first reading拉取你的第一份读数

Needs macOS, a signed-in Claude Code with a Pro/Max subscription, and system python3. Optional: point Claude Code's statusLine at the bridge for zero-cost instant updates while you work. Uninstall anytime with ./uninstall.sh — it never touches your Claude Code credentials or data.需要 macOS、已登录且拥有 Pro/Max 订阅的 Claude Code，以及系统自带的 python3。可选：把 Claude Code 的 statusLine 指向桥接脚本，干活时零成本即时刷新。随时用 ./uninstall.sh 卸载——绝不动你的 Claude Code 凭证和数据。

Why it's different它有何不同

Small, honest, and out of your way小巧、诚实、不挡道

It only speaks up when it matters — and only ever tells the truth.只在要紧时出声，而且只说实话。

Glance and go一眼就懂

When you have room it's just a quiet number. As your quota runs low it turns amber, then red, with a reset countdown — tucked in the corner of your menu bar, notch-friendly and always out of your way.宽裕时它只是一个安静的数字；额度吃紧，它自己变橙、变红，带上重置倒计时。安静待在菜单栏角落，刘海友好，从不挡你的道。

Never lies to you绝不骗你

If the numbers go stale it honestly fades to gray — never old data dressed up as live. It either tells you the truth or admits it's out of date.数据过期就老实变灰，绝不拿旧数字伪装成实时。要么给你真话，要么明说自己旧了。

Costs you nothing零额度消耗

It refreshes quietly in the background and never spends a token of your subscription — and keeps working even when you've stepped away from your desk.后台静默刷新，永不花掉你订阅里的一个 token；你离开座位，它也照常工作。

How it works工作原理

Three small layers三个小分层

Each does one job. The render layer never touches the network when the background layer is healthy.每层只干一件事。后台层健康时，渲染层从不碰网络。

01 RENDER渲染

The gauge表盘

plugin/claude-gauge.15s.sh

A SwiftBar plugin that wakes every 15 seconds, reads whichever of live.json or cache.json is newer, and draws the status-aware title and dropdown. Pure rendering — no secrets, no API calls in the happy path.一个 SwiftBar 插件，每 15 秒醒来一次，读取 live.json 和 cache.json 中较新的那个，绘制随状态变色的标题和下拉。纯渲染——正常路径下不碰任何密钥，不发任何 API 请求。

02 DATA数据

The refresher刷新器

refresher/claude-gauge-refresh.sh

A LaunchAgent (dev.earthonline.claude-gauge) fires every 30s and adaptively throttles — 45s when critical, 240s when idle to dodge 429s. It polls the usage API, atomically writes the cache, and self-heals the token via a zero-cost OAuth refresh.一个 LaunchAgent（dev.earthonline.claude-gauge）每 30 秒触发一次，并自适应节流——告急时 45 秒，闲置时 240 秒，以躲开 429 限流。它轮询用量 API，原子地写入缓存，并通过零成本的 OAuth 刷新给 token 续命。

03 BRIDGE桥接 optional可选

The statusLinestatusLine 桥接

bridge/claude-gauge-statusline.py

Wired as Claude Code's statusLine command, it reads the rate limits CC already hands its status bar and writes live.json — so the menu bar updates instantly while you work, with no API call and no token needed. Applies to new sessions started after you configure it.作为 Claude Code 的 statusLine 命令接入，它读取 CC 已经喂给自己状态栏的限额，写入 live.json——这样你干活时菜单栏即时更新，无需任何 API 请求，也不用 token。只对配置之后新开的会话生效。
